# Code-Quiz
A quiz to test you knowledge! About what? Take it and find out!

## User Story

```
AS A coding boot camp student
I WANT to take a timed quiz on JavaScript fundamentals that stores high scores
SO THAT I can gauge my progress compared to my peers
```

## Acceptance Criteria

```
GIVEN I am taking a code quiz
WHEN I click the start button
THEN a timer starts and I am presented with a question
WHEN I answer a question
THEN I am presented with another question
WHEN I answer a question incorrectly
THEN time is subtracted from the clock
WHEN all questions are answered or the timer reaches 0
THEN the game is over
WHEN the game is over
THEN I can save my initials and set my score on the board
```

## Functionality
```
WHEN I OPEN the page I am presented with: a(n) Welcome Statement and a(n) Start Button.
THEN I CLICK-ON the Start Button.
WHEN I CLICK-ON the Start Button I am presented with: a(n) Game Title, a(n) Timer, a(n) Score, a(n) Question, and Choice(s). 
THEN the Timer Starts.
IF the Timer runs-out.
THEN I am presented with: a(n) Score, a(n) Text Box, and a(n) Submit Button.
WHEN I CLICK-ON a(n) Choice I am presented with: a(n) Wrong/Right message and a(n) Next button.
Then I ClICK-ON the Next Button.
WHEN I CLICK-ON the Next Button I am present with: the next question.
IF I am WRONG the remaining time is reduced. 
IF I am RIGHT the remaining time is unaffected.
THEN I am presented with: a(n) new question.
WHEN I finish I am presented with: a(n) Score, a(n) Text Box, and a(n) Submit Button.
THEN I INPUT my name and I CLICK-ON the Submit Button.
WHEN I CLICK-ON the Submit Button I am presented with: a(n) Score Board and a(n) Play Again Button.
THEN I CLICK-ON the Play Again Button.
WHEN I CLICK-ON the Play Again Button then I am presented with a(n) Welcome Statement and a(n) Start Button.
```
